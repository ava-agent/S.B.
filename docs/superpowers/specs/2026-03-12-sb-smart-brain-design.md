# S.B. Smart Brain — MVP 设计规格

## 产品概述

S.B. Smart Brain 是一款 AI 思维陪练工具。用户每天和一个「街头杠精」风格的 AI 进行 3 轮文字辩论，辩论结束后生成评分报告和毒舌点评，可保存为分享卡片传播。

核心卖点：名字的双关就是产品——**S.B. 帮你不再 SB。**

## MVP 范围

### 包含
- 每日一辩：系统每天推荐一个争议话题
- 用户选立场（正方/反方）
- 与 S.B. 进行 3 轮纯文字辩论
- 辩论结束生成评分报告 + 毒舌点评
- 生成可保存的分享卡片图片
- 匿名使用，无需登录

### 不包含
- 用户自定义话题
- 账号系统 / 登录注册
- 段位系统 / 历史档案
- 语音辩论
- 多人对战

## 页面结构

单页应用，三个状态切换：

### 状态 1：首页（今日辩题）
- 展示今日辩题标题
- 正方 / 反方两个按钮供用户选择立场
- 底部「往期辩题」入口（可选，MVP 可省略）

### 状态 2：辩论中（聊天界面）
- 顶部：返回按钮 + 辩题 + 当前轮次 (1/3)
- 中部：对话流，S.B. 深色气泡 + 用户浅灰气泡
- 底部：输入框 + 发送按钮
- 每轮用户发送后，等待 S.B. 回复，轮次 +1
- 3 轮结束后自动进入报告状态

### 状态 3：报告（评分 + 分享卡片）
- 辩题 + 用户立场
- 四维评分展示（逻辑严密度、论据质量、情绪控制、反驳能力）
- SB 指数（0-100 进度条）
- 毒舌点评（一句话）
- 操作：保存卡片 / 再来一局

## AI 辩论引擎

### 辩论流程

```
用户选立场
  ↓
第 1 轮：S.B. 先手开杠（挑衅式立论，故意激怒）
  ↓ 用户回复
第 2 轮：S.B. 逐句拆解用户论点（抓逻辑漏洞，升级攻击）
  ↓ 用户回复
第 3 轮：S.B. 终极反驳（抛出用户没考虑过的角度）
  ↓ 用户回复
  ↓
S.B. 生成评分报告 + 毒舌点评
```

### S.B. 人格设定

| 维度 | 设定 |
|---|---|
| 身份 | 街头杠精，为杠而生，但逻辑功底扎实 |
| 语气 | 挑衅、讽刺、接地气，像微博评论区最能杠的那个人 |
| 策略 | 第 1 轮挑衅立场，第 2 轮抓逻辑漏洞，第 3 轮降维打击 |
| 底线 | 不人身攻击用户本人，只攻击论点；不涉及政治敏感 |
| 长度 | 每轮回复 150-250 字 |

### 评分体系

| 维度 | 权重 | 评判标准 |
|---|---|---|
| 逻辑严密度 | 35% | 论证链是否自洽，有无逻辑谬误 |
| 论据质量 | 30% | 是否有事实/数据支撑，还是纯观点输出 |
| 情绪控制 | 20% | 是否被激怒后开始人身攻击或偏离论题 |
| 反驳能力 | 15% | 是否有效回应了 S.B. 的攻击点 |

评分等级：S / A+ / A / A- / B+ / B / B- / C+ / C / D

### SB 指数

```
SB 指数 = 100 - 加权总分
```

| 区间 | 评语 |
|---|---|
| 0-20 | 「逻辑鬼才，杠不动你」 |
| 21-40 | 「思路清晰，有两把刷子」 |
| 41-60 | 「还行，但漏洞不少」 |
| 61-80 | 「建议多读点书」 |
| 81-100 | 「经典 SB 发言，建议闭嘴」 |

### 毒舌点评

AI 额外生成一句话总评：
- 必须具体引用辩论中用户的某个论点或行为
- 必须有比喻或类比
- 控制在 30 字以内

示例：
- 「论据像方便面调料包——有是有，就是太单薄」
- 「第二轮破防了吧？打字都能感觉到你在抖」
- 「逻辑链断得比耳机线还快」

## 数据模型

### topics（每日辩题库）

| 字段 | 类型 | 说明 |
|---|---|---|
| id | UUID, PK | |
| title | TEXT | 辩题标题 |
| description | TEXT | 背景补充（可选） |
| publish_date | DATE | 发布日期，每天一题 |
| category | TEXT | 分类（社会/科技/生活/哲学） |
| created_at | TIMESTAMPTZ | |

### debates（辩论记录）

| 字段 | 类型 | 说明 |
|---|---|---|
| id | UUID, PK | |
| topic_id | UUID, FK → topics | |
| stance | TEXT | 'for' \| 'against' |
| messages | JSONB | 完整对话 [{role, content, round}] |
| score_logic | TEXT | 逻辑严密度评分 |
| score_evidence | TEXT | 论据质量评分 |
| score_emotion | TEXT | 情绪控制评分 |
| score_rebuttal | TEXT | 反驳能力评分 |
| sb_index | INTEGER | SB 指数 0-100 |
| roast | TEXT | 毒舌点评 |
| created_at | TIMESTAMPTZ | |

Supabase 使用策略：
- 不用 Auth，不用 RLS（MVP 数据不敏感）
- API 通过 Next.js Server Action 调用，不暴露 key
- topics 手动 seed，debates 辩论结束后一次性写入

## 分享卡片

### 内容结构
1. 品牌标识：S.B. Smart Brain
2. 辩题 + 用户立场
3. 三维评分卡片并排（逻辑 / 论据 / 情绪）
4. SB 指数进度条（亮绿色）+ 评语
5. 毒舌点评（斜体带引号）
6. CTA：「来和 S.B. 杠一杠 →」+ 网址

### 设计规范
- 尺寸：1080×1350px（Instagram / 小红书友好）
- 白色背景，亮绿强调色
- 使用 Satori (@vercel/og) 服务端生成

### 传播机制
- SB 指数低 → 炫耀分享
- SB 指数高 → 自嘲分享
- 两种结果都有传播动力
