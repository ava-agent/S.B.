# S.B. — Smart Brain ⭐

> 来源：争议命名AI产品创意集（2026-03-11）

## 产品概述
- **AI 思维陪练，帮你在说蠢话/做蠢事之前发现问题**
- 名字的双关就是产品：**S.B. 帮你不再 SB。**

## 产品定位
AI 思维陪练工具。

## 三大核心场景

### 场景 1：发言前检查（防社死）
```
用户输入：准备发的微博 →「中医就是伪科学」

S.B. 输出：
  ⚠️ SB指数：72/100（高危）

  你的论证漏洞：
  1. 以偏概全 — 中医包含经验医学和哲学体系，不能一刀切
  2. 缺少定义 — 你没有定义"伪科学"的标准
  3. 预判反驳 — 对方会拿屠呦呦青蒿素反驳你

  改进建议：→「中医的部分理论缺乏现代循证医学验证」
```

### 场景 2：决策压力测试（防踩坑）
```
用户输入：我想裸辞去学AI

S.B. 输出：
  💰 财务：你的存款能撑几个月？
  📉 时机：AI岗位竞争白热化，半年后还是风口？
  🎯 替代方案：为什么不能边工作边学？
  🧠 认知偏差：你可能存在「幸存者偏差」

  综合SB指数：45/100（中等风险）
```

### 场景 3：每日思维训练（变聪明）
```
每日一题：「应不应该禁止短视频？」

选立场 → S.B. 站对面 → 来回辩论 3 轮

结束后生成报告：
  逻辑严密度：B+
  论据质量：A-
  情绪控制：C（第2轮开始人身攻击了）
  累计段位：白银 III → 白银 II ↑
```

## 病毒传播机制

### SB 指数分享卡片
```
┌─────────────────────┐
│  🧠 我的SB指数：23   │
│  逻辑力：A+         │
│  信息素养：B         │
│  情绪控制：A         │
│  段位：铂金 I        │
│  超过了87%的用户     │
│  ── Smart Brain ──  │
│  测测你的SB指数 →    │
└─────────────────────┘
```

传播动力：
- 分数低 = 炫耀「我不SB」→ 分享
- 分数高 = 自嘲「我果然SB」→ 也分享
- 两种结果都有传播动力

### 社交挑战
- 「我觉得你这个观点很SB，让S.B.来评评理」→ 分享给朋友
- 双人对战辩论，S.B. 当裁判打分

### 名字本身就是传播
- 「我每天用S.B.训练思维」
- 「S.B.说我这个决定有问题」
- 媒体标题：「S.B. 获百万用户」

教育市场机会：

- 公务员申论备考
- 企业培训（决策力提升）

## MVP 技术方案

一周可上线的最小版本：单页应用，输入观点/决策 → 输出 SB 指数 + 分析 → 生成分享卡片

```
技术栈：
- 前端：Next.js + TailwindCSS
- 后端：Claude/GPT API + 精心设计的 system prompt
- 部署：Vercel
- 核心工作量：prompt engineering
```

## 关键成功因素

1. **AI 点评质量** — 精准指出具体逻辑问题，给出谬误名称（稻草人谬误、滑坡论证、幸存者偏差...）
2. **语气有性格** — 直接、犀利、偶尔扎心，但始终有道理（不能像 ChatGPT 一样温吞）
3. **SB指数可信** — 基于可解释的维度打分，不是随机数
4. **分享卡片好看** — 传播核心载体，设计感要强
